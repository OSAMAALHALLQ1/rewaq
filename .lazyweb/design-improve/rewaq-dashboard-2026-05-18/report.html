تحسين تصميم رواق

تقرير Lazyweb مختصر، 2026-05-18.

المراجع

  • Wise Business: حالة عامة وإجراءات أولية واضحة.
  • Buffer: منشور واحد وقنوات متعددة وحالة نشر مفهومة.
  • Lightspeed: تشغيل مطاعم يربط المخزون والقائمة والفروع.
  • Ramp/Mercury: بطاقات قرار قبل المخططات التفصيلية.

ما تم تطبيقه

  • تحويل الصفحة الداخلية إلى "اليوم في رواق".
  • إضافة إجراءات مباشرة للبيع والشراء والجرد والنشر.
  • توسيع النشر الاجتماعي عبر Node-RED webhook مع fallback تجريبي.
  • إضافة API داخلي: /api/node-red/social-publish.
